# Chess Board Game

A simple chess board game built with HTML, CSS, and JavaScript.

## Features
- Interactive chess board
- Black and white pieces
- Turn-based gameplay (White's Turn / Black's Turn)
- Click to select and move pieces

## How to Play
1. Click on a white piece to select it
2. Click on the destination square to move
3. Players alternate turns
4. White moves first

## Files
- index.html - Main game file (HTML + CSS + JavaScript)

## How to Run
Simply open index.html in any web browser.
